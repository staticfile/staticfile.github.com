ABOUT THE CDN

我们的目标是提供这样一个仓库，让它尽可能全面收录优秀的开源库，并免费为之提供 CDN 加速服务，使之有更好的访问速度和稳定的环境。同时，我们也提供开源库源接入的入口，让所有人都可以提交开源库，包括 JS、CSS、image 和 swf 等静态文件。

全面

目前像包括 Google Ajax Library 等在内提供的 CDN，使用起来却都有些局限，因为他们只提供了部分 JS 库。

当然，我们还可以有像 CDNJS 这样的平台，存储了大部分主流的 JS 库，甚至 CSS、image 和 swf，但国内的访问速度却不是很理想，并且缺少很多国内优秀开源库。我们希望提供更完整的资源。

提交开源产品

开放

所有人均可通过 Pull Request 提供开源的库，一旦被官方仓库 Merge，则会自动存储到 CDN 上。目前采用人员审核的方式添加，标准是在 Github 上的 star 数超过 100 个。

当然，像放在 Google Code 等仓库上的优秀开源库。我们也会一一添加。希望大家也一起来帮助我们完善这个库，让更多人更好地享受这些开源成果。
